## 项目介绍

**电子科技大学2020级《计算机网络与通信技术》课程的作业与项目源码。**

### 作业简介

1. 词汇解释；
2. 习题；
3. 小结思维导图；
4. 讨论题PPT与讲稿。

### 项目简介

1. 基于Python，进行最小网元设计；
2. 基于Packet Tracer，实现简单的校园互联网；
3. 基于WireShark，对网络流量抓包分析；
4. 基于Simulink，完成数字数据的调制解调。
